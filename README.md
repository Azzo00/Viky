# Time Tracker

App web per tracciare le ore di lavoro per attività e commessa.

## Funzionalità

- Aggiungi attività con descrizione, commessa e ore obiettivo
- Timer per ogni attività (un timer alla volta)
- Barra di progresso colorata per commessa
- Storico sessioni per ogni attività
- Esportazione CSV
- Dati salvati in localStorage (persistenti tra sessioni)

## Utilizzo

Apri semplicemente `index.html` nel browser — nessuna dipendenza, nessun server richiesto.

## Deploy rapido su GitHub Pages

1. Crea un repo su GitHub
2. Carica `index.html`
3. Vai su **Settings → Pages → Branch: main → Save**
4. L'app sarà disponibile su `https://<username>.github.io/<repo>/`
